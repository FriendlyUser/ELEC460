2.1 Steady-state error

2.1.1 Definition

Consider a feedback system as in Fig. ??.

The steady-state error (if it exists) is defined by

\( \seteqsection {2} \)

\begin{equation} \label {eq:sse} e_\sse := \lim _{t\to \infty } e(t). \end{equation}

If the system is stable, one can apply the Finaly Value Theorem to obtain

\[ \lim _{t\to \infty } e(t) = \lim _{s\to 0} sE(s). \]

In general, one would like the steady-state error to be as small as possible, ideally zero.

2.1.2 System type and steady-state error

However, if the system is stable and unity-feedback, then one can determine the system type and use this information to obtain the steady-state error much more easily. This is a margin note using the geometry package, set at 5cm vertical offset to the first line it is typeset. Let \(G\) be the feedforward transfer function. Assume that \(G\) is written as

\[ G=\frac {N(s)}{s^kD(s)}, \]

where \(N\) and \(D\) are not factorizable by \(s\). Then the type of the system is given by the integer \(k\). Fig. ?? shows the general form of systems of types 1, 2 and 3.

Let us carry further the computation of the steady-state error for a stable unity-feedback system.

\[ sE(s) = s(R(s)-C(s)) = s\left (R(s)-\frac {G(s)}{1+G(s)}R(s)\right )=\frac {s}{1+G(s)}R(s). \]

For unit-step input, one has \(R(s)=1/s\). Thus,

\[ sE(s) = \frac {s}{1+G(s)}*\frac {1}{s} = \frac {1}{1+G(s)} \xrightarrow [s\to 0]{}\frac {1}{1+\lim _{s\to 0}G(s)}. \]

For a system of type 0, one has

\[ \lim _{s\to 0}G(s) = \lim _{s\to 0}\frac {N_0(s)}{D_0(s)} = \frac {N_0(0)}{D_0(0)}. \]

Thus, the steady-state error of a system of type 0 for unit-step input is given by

\[ e_\sse = \frac {1}{1+\frac {N_0(0)}{D_0(0)}}. \]
